# Gerador de Aulas Científicas

Aplicativo para geração de aulas científicas a partir de PDFs, com recursos avançados de análise e síntese.

## Instalação no Celular

### Método 1: Instalação via GitHub Pages

1. Acesse o aplicativo pelo link: https://seu-usuario.github.io/gerador-aulas-app/
2. No navegador, toque no menu (três pontos) e selecione "Adicionar à tela inicial" ou "Instalar aplicativo"
3. Confirme a instalação

### Método 2: Instalação Local

1. Baixe este repositório para seu computador
2. Inicie um servidor HTTP local na pasta do projeto
3. Acesse pelo celular usando o IP do seu computador na rede local
4. No navegador do celular, instale como PWA conforme instruções acima

## Recursos

- Upload e análise de PDFs científicos
- Geração automática de aulas estruturadas
- Sistema de autenticação para salvar suas aulas
- Preferências personalizáveis
- Biblioteca de aulas salvas
- Funciona offline após instalação

## Requisitos Técnicos

- Navegador moderno com suporte a PWA
- Conexão com internet para o primeiro acesso
